## 作用：
* 能够实时显示选择的客户端所建立的每一条网络连接。
* 能够解析并查看https(暂不开源)和http的包内容。
* 默认对抓包数据进行保存，并可查看保存的记录。
* 能够自动将抓包数据进行解析，并保存。
* 支持https抓包(暂不开源)，支持图片、GZIP、GIF等格式的解析。
* 支持选择对应的APP进行抓包。
